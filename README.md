# 🥬 Saral Mandi AI – Smart AI-Powered Agriculture Marketplace

Saral Mandi AI is an intelligent agriculture marketplace platform designed to empower farmers, buyers, and local vendors through AI-driven insights, smart crop recommendations, multilingual support, and modern digital market access.

The platform simplifies agricultural trading and improves farmer profitability using artificial intelligence, real-time analytics, and accessible technology.

GitHub Repository: [Saral Mandi AI Repository](https://github.com/DemocratiseResearch/SARAL?utm_source=chatgpt.com)

# ⚙ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/santhosapriyan-eng/saral-mandi-ai.git

cd saral-mandi-ai
```

---

## 2️⃣ Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000

MONGO_URI=your_database_url

GEMINI_API_KEY=your_gemini_api_key

WEATHER_API_KEY=your_weather_api_key
```

---

# ▶ Running the Project

## Start Backend

```bash
cd backend
npm start
```

## Start Frontend

```bash
cd frontend
npm run dev
```

Open in browser:

```bash
http://localhost:5173
```
